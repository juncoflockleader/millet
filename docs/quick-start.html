Setup Checklist

Progress is saved in this browser.

0 of 6 complete

1. Run tests

Start with the deterministic suite so you know the local repo is healthy.

2. Start demo

Run the hotseat server and open the Ember Duel browser UI.

3. Take an action

Click a card or hero to select it, then click or drag it onto a highlighted target. Action buttons still work for explicit commands.

4. Inspect UI data

Open Preview for fixture states, Assets for manifest previews and import drafts, Cards for template drafts, behavior text sync, and property badges, and Layout for the viewport-fitted board editor with document fields, presets, diagnostics, snap, copy, flip, fill, and authored region guides.
